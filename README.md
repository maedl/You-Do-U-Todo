## Requirements

- Add tasks

- Able to mark tasks as done
- Tasks has due dates
- Completed tasks goes at the bottom of the list
* Tasks are visually marked when: 
  - Expired
  - Due in 5 days
- Display unique icons on tasks depending on selected category  
* Tasks can be sorted by:
  - Due date
  - Title
  - Date added
